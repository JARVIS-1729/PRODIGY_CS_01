# PRODIGY_CS_01

Task 01: Caesar Cipher

A procedural Python script that implements one of the oldest encryption algorithms in history.

Concept: Shift-based cryptography and ASCII manipulation.

Features: * Encrypts and decrypts user-provided text.

Preserves spaces, numbers, and special characters.

Uses modulo arithmetic (% 26) to wrap characters smoothly across the alphabet.

Tech Stack: Python (Built-in libraries)
